Or Audit Your Own Data

The data file should contain:
  • binary assessments (0 or 1 for each row) made by the predictive model (1 denotes the individuals selected for the intervention)
  • columns containing attributes you want to audit for bias
  • true future outcome for each individual (if you want to audit bias based in disparate errors)
Please note that any data you upload will be used to generate the audit report for you and then deleted. We do encourage you to use the desktop version of the audit tool if your data is private and sensitive.
